# Zeno — AI 原生的购物入口

> 用一场对话，替代一整面货架。
> Describe your need, not a brand.

Zeno 是一个 **AI 原生的购物市场（AI-native marketplace）**，目标市场为**中国 + 美国**。它的核心主张只有一句：**用 AI 替代货架**。

传统电商把海量 SKU 铺成一面无限长的货架，让用户自己去搜、去比、去挑——决策成本高、信息过载、容易被营销操纵。Zeno 反过来：用户用自然语言**描述需求**（场景、约束、隐性偏好），AI 理解需求、跨品牌跨品类推理、收窄到最适合的少数选项并**解释为什么推荐**。

Zeno 追求**需求端与供给端的双高效**：

- **需求端高效**：跟 AI 对话找到最适合自己的商品。**描述需求，而不是描述品牌。** 把"挑选"这件认知苦力从用户转移给 AI。
- **供给端高效**：**先**通过 CPS 联盟（淘客 / 京东联盟 / 多多进宝 / 抖音 / Amazon Associates / CJ / Rakuten / Impact 等）瞬间、轻资产地接入已经存在的全网高质量商品库；**之后**通过 **0 佣金**把商家直接引入平台，沉淀自有供给。

## 🎯 一页纸速览（Executive Summary）

**是什么**：Zeno 是一个中立的、需求驱动的购物大脑（demand-side brain）＋ 双边高效的市场（two-sided marketplace）。用户对话购物，AI 帮你从全网找到"最适合你"的商品，而不是"广告出价最高"的商品。

**为什么是现在（2026）**：
- AI 购物已成习惯——ChatGPT 周活 9 亿、Amazon Alexa for Shopping 服务 3 亿+ 用户、2025 假期 AI 导流增长约 700% 且转化率高出 31%（Adobe Analytics）。
- agentic checkout 基础设施成熟——Stripe/OpenAI 的 **ACP**、Visa Trusted Agent Protocol、Mastercard Agent Pay 让"AI 直接下单"成为可能。
- 中国 CPS 生态极度成熟——淘客 / 京东联盟 / 多多进宝 / 抖音电商，可瞬间接入全网商品库，**轻资产冷启动**。
- 格局有空隙——货架平台自营 AI（利益不中立、只推自己货架）、通用助手（购物是边角功能、供给浅、中国本地化弱），**没人专注做"中立 + 需求驱动 + 双边高效"**。

**怎么做**：
- 供给侧两阶段：**CPS 接入（轻资产、即时有收入）→ 0 佣金入驻（沉淀自有供给）**。
- 需求侧一个杀手级体验：**描述需求 → AI 收窄 → 解释推荐 → 一键履约**。
- 护城河：①商品知识图谱 + 需求理解模型；②双边网络效应；③中立信任（不自营、披露利益、可解释）；④中美双市场本地化。

**怎么赚钱**：阶段一靠 CPS 佣金分成；阶段二靠**非佣金**货币化（履约 / 营销工具 / 数据洞察 / 会员 / agentic checkout 支付分成），刻意避免与商家 GMV 对立，守住中立。详见 [08](./08-business-model.md)。

## 🧭 设计原则（贯穿所有文档）

1. **需求优先于供给（Demand-first）**：一切从"用户想解决什么问题"出发，而不是"我们有什么货要卖"。
2. **中立是核心资产（Neutrality is the moat）**：Zeno 的推荐必须可信。任何与中立冲突的货币化都要克制并披露。
3. **轻资产、快接入（Asset-light）**：不自建库存与物流；用 CPS 和 agentic checkout 站在巨人肩膀上。
4. **可解释（Explainable）**：每个推荐都能回答"为什么是它、为什么不是别的"。
5. **真实性高于流畅性（Factuality > fluency）**：价格、库存、参数、可购买性必须真实，宁可少说不可幻觉。
6. **双市场同构、本地化分支（One core, localized edges）**：核心引擎统一，履约 / 合规 / 文化在边缘做本地化。

## 📖 术语表（Glossary）

| 术语 | 含义 |
|------|------|
| **货架（Shelf）** | 传统电商以 SKU/品牌为单位、由用户自行检索筛选的商品陈列范式。Zeno 要替代的东西。 |
| **对话式商务（Conversational Commerce）** | 以自然语言对话为主交互的购物范式。 |
| **需求端 / 供给端（Demand / Supply side）** | 双边市场的两侧：买家（需求）与商品/商家（供给）。 |
| **CPS（Cost Per Sale）** | 按成交付费的联盟营销模式。Zeno 供给侧冷启动的核心。 |
| **联盟 / 淘客（Affiliate / 淘客）** | 提供商品库与成交佣金的联盟平台（淘宝联盟、京东联盟、多多进宝、Amazon Associates、CJ、Rakuten、Impact 等）。 |
| **归因（Attribution）** | 将一笔成交追溯到 Zeno 引导的链路（PID / SubID / uid 等），用于结算佣金。 |
| **ACP（Agentic Commerce Protocol）** | OpenAI + Stripe 主导的开放标准，让 AI 代理直接完成下单与支付。 |
| **商品知识图谱（Product Knowledge Graph）** | 把异构商品富化成可推理的结构化属性、场景、关系网络。 |
| **需求满足率（Need-Fit Rate）** | 衡量"AI 是否真的找到了最适合用户"的核心质量指标。详见 [11](./11-metrics.md)。 |
| **混合 UI（Hybrid UI）** | 对话流 + 动态生成的商品卡片 / 对比表 / 筛选器的融合界面。 |
