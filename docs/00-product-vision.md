## 1. 一句话定位

**Zeno 是 AI 原生的购物入口——用对话替代货架，从全网为你找到"最适合你"的商品。**

- 对用户：**描述需求，而不是描述品牌。**
- 对市场：一个**中立的需求大脑** ＋ 一个**双边高效的市场**。

---

## 2. 愿景、使命、价值观

**愿景（Vision）**
> 让每一次购买决策，都像问一个真正懂行、且只为你着想的朋友。

**使命（Mission）**
> 用 AI 消灭购物中的信息不对称与认知苦力，让需求与供给以最低摩擦相遇。

**价值观（Values）**
1. **站在买家一边（Buyer-aligned）**——当商业利益与用户利益冲突时，选用户。
2. **真实胜于动听（Truth over polish）**——不夸大、不幻觉、不诱导。
3. **轻装上阵（Asset-light）**——不重复造轮子，站在成熟基础设施的肩膀上。
4. **可解释的智能（Explainable intelligence）**——黑箱推荐不可信，每个建议都要讲清道理。

---

## 3. 核心洞察：货架电商错在哪里

传统电商（淘宝、Amazon、京东、拼多多）本质是一面**为供给组织、而非为需求组织**的无限货架。它带来三个结构性问题：

### 3.1 用户被迫做"翻译"
用户心里想的是**问题**（"我要给我妈挑一双久站不累、好穿脱、防滑的鞋"），但货架只懂**关键词和品牌**。用户必须把需求翻译成搜索词、再在结果里反向猜测哪些商品满足隐性约束。这层翻译是纯认知损耗。

### 3.2 信息过载 + 决策瘫痪
一次搜索动辄几千上万个结果，参数表晦涩、评论真假难辨、价格被优惠券和满减游戏搞得不透明。用户在"挑选"上花费大量时间与意志力，却仍不确定选对了。

### 3.3 排序被商业利益操纵
货架的排序权掌握在平台手里，而平台的收入来自广告竞价（坑产、直通车、Sponsored）。**用户看到的"最好"，往往是"出价最高"。** 这是货架范式无法自愈的利益冲突。

> **结论**：货架把"组织供给"的效率做到了极致，却把"理解需求"的责任推给了用户。这正是 AI 的机会。

---

## 4. Zeno 的解法：双高效

### 4.1 需求端：从"描述品牌"到"描述需求"

用户用自然语言说出真实需求，Zeno 负责其余一切：

```
用户：「我想要一个能放进通勤背包、续航至少两天、降噪好一点的耳机，
        预算 800 以内，最好别是入耳式的，戴久了耳朵疼。」

Zeno：理解 → 抽取约束（便携/续航≥2天/降噪/价格≤800/非入耳/久戴舒适）
      → 跨品牌检索 → 多目标排序 → 收窄到 2–3 个
      → 解释「为什么是它，以及它的取舍」
      → 一键履约
```

**关键差异**：AI 把"挑选"的认知劳动从用户身上转移走了。用户只需表达和确认，不需要逛。

### 4.2 供给端：两阶段，从"接入全网"到"沉淀自有"

| 阶段 | 做法 | 目的 |
|------|------|------|
| **阶段一：CPS 接入** | 通过淘客 / 京东联盟 / 多多进宝 / 抖音 / Amazon Associates / CJ / Rakuten / Impact 等联盟，接入**已经存在的**全网高质量商品库 | 轻资产冷启动；**第一天就有海量供给和真实收入**；零库存风险 |
| **阶段二：0 佣金入驻** | 用 0 佣金把商家**直接**引入 Zeno，靠非佣金方式（履约/营销工具/数据/会员/支付分成）赚钱 | 沉淀**自有供给**，获得独家商品、更优价格、更深履约，建立护城河 |

> 详见 [供给侧设计](./06-supply-side.md) 与 [商业模式](./08-business-model.md)。

**为什么这个顺序是对的**：先 CPS，让产品在"没有自有供给"的情况下也能给用户**完整体验**（解决冷启动的鸡生蛋问题）；当需求侧跑通、有了流量，0 佣金就成了对商家极具吸引力的 wedge，把供给从"借来的"变成"自己的"。

## 5. 为什么是现在（Why Now, 2026）

四个条件在 2026 年同时成熟，这是 Zeno 的时间窗口：

1. **AI 购物已成习惯**
   - ChatGPT 周活约 **9 亿**；Amazon 把 Rufus 升级为 **Alexa for Shopping**（2026-05），2025 年服务 **3 亿+** 购物者。
   - 2025 假期季，AI 平台导流至零售网站增长约 **700%**，且这些用户转化率比传统搜索高 **31%**（Adobe Analytics）。
   - 用户已经接受"问 AI 买东西"，教育成本大幅下降。

2. **agentic checkout 基础设施成熟**
   - OpenAI + Stripe 的 **Agentic Commerce Protocol (ACP)**、Visa **Trusted Agent Protocol**、Mastercard **Agent Pay** 让"AI 直接、安全地下单支付"成为可能（Shared Payment Token 等）。
   - Zeno 不必自建支付，可直接站在这层之上做履约。

3. **中国 CPS 生态极度成熟**
   - 淘客（PID）、京东联盟、多多进宝（uid 跟单）、抖音 / 美团 / 饿了么 CPS 一应俱全，可**瞬间接入全网商品库**并即时结算佣金。
   - 这让中国市场的"阶段一"几乎零成本启动。

4. **格局有真实空隙**
   - 货架平台自营 AI（淘宝问问、Alexa for Shopping）——**利益不中立**，只推自家货架，跨平台比价能力天然受限。
   - 通用 AI 助手（ChatGPT / Perplexity / Gemini 的购物功能）——购物只是**边角功能**，供给侧靠通用抓取或有限 feed，缺乏深度商品理解、履约和（尤其是中国的）本地化。
   - **没有人专注做"中立 + 需求驱动 + 双边高效"的购物专属产品。** 这正是 Zeno 的位置。

> 详细竞品分析见 [01-市场与竞品分析](./01-market-analysis.md)。

## 6. 护城河（Moats）

Zeno 不靠单点功能取胜，而靠四个相互强化的结构性优势：

### 6.1 商品知识图谱 + 需求理解模型
把全网异构、脏乱的商品数据，富化成**可推理的结构化知识**（属性、适用场景、人群、关系、真实口碑）。这是把"全网商品"变成"AI 能精准匹配的供给"的核心资产，越用越厚。详见 [05](./05-ai-engine.md) 与 [07](./07-data-model.md)。

### 6.2 双边网络效应
```
更多需求 → 更多 GMV → 更多商家愿意 0 佣金入驻
   ↑                                      ↓
更精准匹配 ← 更优质/更独家供给 ← 更好的价格与商品
```
这是经典的双边飞轮，一旦转起来，后来者难以追赶。

### 6.3 中立信任
不自营、披露利益、可解释推荐——Zeno 的品牌资产是"它真的为我着想"。货架平台因自营和广告模式**无法**复制这种中立。这是认知层面的护城河。

### 6.4 中美双市场的本地化履约与合规
统一的需求大脑 + 本地化的供给 / 履约 / 合规边缘。同时跑通两个全球最大电商市场，规模与数据优势是单市场玩家无法企及的。

## 7. 我们明确**不做**什么（Non-Goals）

- ❌ **不自建库存 / 仓储 / 物流**（至少前两阶段）——保持轻资产。
- ❌ **不做内容种草社区**——Zeno 是"决策与成交"，不是"逛和刷"。种草是上游，不是我们的主战场。
- ❌ **不靠竞价广告作为主要排序信号**——这会摧毁中立性这一立身之本。
- ❌ **不追求 SKU 数量的虚荣指标**——我们追求"需求满足率"，不是"商品总数"。
- ❌ **不做通用 AI 助手**——Zeno 专注购物这一个垂直，把它做到极致。

## 8. 战略叙事（给不同对象的一句话）

- **给用户**：别再逛了。告诉 Zeno 你要什么，它帮你从全网挑好、讲清楚、买到手。
- **给商家**：0 佣金入驻，把对的商品精准送到有真实需求的人面前。
- **给投资人**：AI 购物时代的中立入口与双边市场——轻资产冷启动、双边网络效应、中美双市场。
- **给团队**：我们在为买家重建购物的信任。每个决策都问一句：这是为用户好，还是为我们好？

## 9. 北极星

**北极星指标：被满足的真实需求量（Fulfilled Needs）**——即"用户表达需求 → Zeno 给出推荐 → 用户认可并完成购买且未后悔"的闭环数量与质量。

它同时约束需求端（匹配得准不准）、供给端（有没有合适的货）、和信任（用户是否后悔）。具体定义与指标体系见 [11-度量体系](./11-metrics.md)。

## ⚖️ 正反方辩论（Red Team / Blue Team）

> 对本文关键主张做一次对抗性审视——正方力挺、反方进攻，最后给出裁决与"会改变主意的信号"。

**议题：用 AI 对话替代货架、做中立双边市场，是真命题与真时机吗？**

**🟦 正方（支持）**
- 货架的"竞价排序 vs 用户利益"是结构性冲突，自营巨头改不动——这是 Zeno 可长期占据的空位。
- LLM 已让"描述需求"真正可行，CPS 让轻资产冷启动成立，时机窗口确实打开。
- 中立是认知层护城河：用户一旦相信"它为我着想"，迁移成本高。

**🟥 反方（质疑）**
- Amazon / Google 在 AI 购物上有数据、履约、流量的碾压级优势，且能用补贴模仿"中立"。
- 多数购物是高频、低决策、冲动或复购，本就不需要对话——对话购物可能只是小众高决策场景。
- "替代货架"叙事过大，高估了用户改变十年习惯的意愿。

**⚖️ 裁决**
- 不正面硬刚货架全场景，聚焦高决策、高信息不对称品类（对话价值最大、巨头中立短板最痛）。
- 中立靠制度化 + 垂直专注建立，正是巨头因既得利益难以复制之处。
- **Kill 信号：** 在选定品类，AI 推荐的采纳率 / 需求满足率长期不显著高于货架搜索，且巨头中立度追平。
