# 08 · 商业模式与增长

> 关联文档：[供给侧](./06-supply-side.md) · [市场分析](./01-market-analysis.md) · [合规](./09-trust-safety-compliance.md) · [度量](./11-metrics.md)
> 本文数字均为**示意假设**（标"（假设）"），用于说明结构，需以真实数据校准。

---

## 1. 核心信条：靠"撮合效率"赚钱，不靠"对商家抽税"

货架平台的收入主要来自**对商家的广告竞价/佣金**，这与"给用户看最好的商品"天然冲突——排序被出价绑架。Zeno 刻意走另一条路：

> **收入来自"帮买卖双方更高效相遇"，而非"对商家 GMV 抽税"。**
> 这样平台利益与用户利益对齐，中立性才可持续（[09](./09-trust-safety-compliance.md)）。

这条信条决定了下面所有收入设计：**任何与排序中立冲突的变现都被克制或排除。**

---

## 2. 收入结构（分阶段）

### 阶段一：CPS 佣金分成（即时、轻资产）
- **机制**：用户经 Zeno 引导成交，联盟支付佣金（Amazon 1–10%、CJ 15–25%、Impact 20–40%、中国淘客/京东/多多按类目）。
- **优点**：第一天就有收入、零库存、与"帮用户买到对的东西"完全对齐（用户买得越满意越成交）。
- **注意**：佣金率**不进入排序**（[07](./07-data-model.md)、[09](./09-trust-safety-compliance.md)）——我们靠"匹配得准→成交多"赚钱，不靠"推高佣金的"。

### 阶段二：非佣金货币化（沉淀自有供给后）
0 佣金把商家引入后，靠**不与 GMV 直接对立**的方式变现：

| 收入线 | 说明 | 与中立的关系 |
|--------|------|--------------|
| **B1 履约/物流增值** | 仓配、客服、退换托管等按使用付费 | ✅ 中立（服务费，与排序无关） |
| **B2 经营/营销工具** | 数据洞察、A/B、需求标签优化、素材工具（订阅） | ✅ 中立（工具费） |
| **B3 会员 / Plus（需求侧）** | 高级比价、降价监控、专家级对比、优先客服 | ✅ 中立（向买家收费，利益更对齐） |
| **B4 支付/履约分成（ACP）** | 站内 agentic checkout 直购的交易服务费（参考 OpenAI Instant Checkout 4%） | ✅ 中立（按成交的服务费，非排序加权） |
| **B5 品牌曝光位（克制）** | **明确标注**的赞助位，**与自然推荐区隔离、不污染排序** | ⚠️ 受限：须显著披露、占比设硬上限、绝不混入"为你推荐"（[09](./09-trust-safety-compliance.md)） |

> **红线**：B5 永远是"标注清楚的广告"，不是"伪装成中立推荐的广告"。一旦广告侵蚀自然推荐的可信度，就动摇了立身之本——宁可少赚。

## 3. 单位经济（Unit Economics，示意）

以单次"被满足的需求"为单位（数字为假设，仅示意结构）：

```
一次成功撮合（假设·中国数码品类）
  到手价 GMV ............................ ¥900
  联盟佣金率(加权) .....................   4%
  毛佣金收入 ...........................  ¥36
  减：AI 推理成本(分级模型+缓存) .......  ¥1–3
  减：归因/对账/基础设施分摊 ...........  ¥1–2
  ─────────────────────────────────────
  阶段一贡献毛利 .......................  ~¥31–34/单
```

**杠杆点**
- **AI 成本**：分级模型（Haiku 级做高频轻任务）、提示/结果缓存、富化离线化 → 单次对话成本受控（[05](./05-ai-engine.md)、[04 §3](./04-system-architecture.md)）。
- **转化质量**：需求理解越准 → 采纳率/转化越高 → 单位获客摊薄越好（AI 导流转化比传统搜索高 ~31%，对我们有利）。
- **复购/记忆**：个性化记忆提升复购，拉低 CAC、提高 LTV。
- **阶段二**：自有供给 + 非佣金收入线叠加，单位经济结构改善且更抗联盟降佣。

**LTV/CAC（假设）**：北极星是"被满足的真实需求量"，高满意→高复购→高 LTV；中立信任降低 CAC（口碑驱动）。目标 LTV/CAC ≥ 3（行业健康线，待验证）。

```

        │ 口碑/复购，CAC↓
        ▼
```

## 5. 中美货币化差异

| 维度 | 🇨🇳 中国 | 🇺🇸 美国 |
|------|---------|---------|
| **阶段一主力** | 淘客/京东/多多/抖音 CPS；价格敏感、到手价玩法重 | Amazon/CJ/Rakuten/Impact；佣金率高、客单价高 |
| **会员意愿** | 偏低，需强价值（比价/返利/省钱） | 较高，订阅文化成熟（Plus 更易成立） |
| **支付/ACP** | 微信/支付宝生态；站内直购需本地化 | ACP/卡组织推进快，B4 更易落地 |
| **广告位（B5）** | 需严守广告法，披露与用语红线 | 需严守 FTC 披露 + AI 披露 |

> 货币化节奏按市场差异排布：中国先吃 CPS + 省钱型会员；美国可更早试 ACP 直购分成与 Plus。

## 6. 为什么这个模型能赢

1. **轻资产**：不压库存、不建物流，CPS + ACP 站在成熟基础设施上。
2. **利益对齐**：主要收入不与用户利益对立 → 中立可持续 → 信任品牌 → 低 CAC。
3. **抗风险**：多联盟 + 多收入线 + 自有供给，单一平台降佣不致命。
4. **网络效应**：双边飞轮 + 两项复利资产，护城河随规模加深。
5. **双市场**：核心引擎复用，规模与数据优势超越单市场玩家。

## 7. 关键财务假设需校准的点（MVP 阶段验证）

- 单次对话真实 AI 成本与缓存命中率。
- 采纳率、购买转化、客单价、加权佣金率（分品类/市场）。
- 复购率与记忆带来的 LTV 提升。
- 0 佣金入驻的渗透速度与非佣金收入线的 ARPU。
- B5 广告位在"不伤中立"前提下的可承受占比上限。

> 这些数字一旦由真实数据替换，本文的单位经济与飞轮假设即可转为可信的财务模型（[度量](./11-metrics.md)）。

## ⚖️ 正反方辩论（Red Team / Blue Team）

> 对本文关键主张做一次对抗性审视——正方力挺、反方进攻，最后给出裁决与"会改变主意的信号"。

**议题：靠"撮合效率"而非"对商家抽税"、非佣金变现 + 克制广告，能撑起商业吗？**

**🟦 正方（支持）**
- 收入不与用户利益对立，使中立可持续、口碑驱动降 CAC。
- 多收入线（履约 / 工具 / 会员 / ACP）抗单一联盟降佣风险。

**🟥 反方（质疑）**
- 非佣金各条线都未验证，可能 ARPU 太低撑不起 AI + 获客成本。
- 放弃广告 / 竞价等于放弃电商最肥的收入池，估值故事偏弱。
- CPS 毛利被平台掐住，不可控。

**⚖️ 裁决**
- 阶段一用 CPS 验证单位经济；阶段二逐条验证非佣金线 ARPU，跑通一条扩一条。
- 广告只做"标注清晰、不污染排序"的克制版作为补充，绝不动排序中立。
- **Kill 信号：** 所有非佣金线 ARPU 都无法覆盖 AI + 获客成本，且 CPS 毛利持续被压缩。
