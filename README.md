# Zeno

> 用一场对话，替代一整面货架。 · **Describe your need, not a brand.**

**Zeno** 是一个 AI 原生的购物市场（AI-native marketplace），目标市场为 **中国 + 美国**。
核心主张：**用 AI 替代货架**——用户用自然语言**描述需求**（场景、约束、隐性偏好），AI 从全网找到"最适合你"的商品、解释为什么推荐、并一键履约；而不是让你在无限货架上自己搜、比、挑。

Zeno 追求**需求端与供给端的双高效**：

- **需求端高效**：跟 AI 对话找到最适合自己的商品。**描述需求，而不是描述品牌。**
- **供给端高效**：**先**通过 CPS 联盟接入已存在的全网高质量商品库（轻资产冷启动），**之后**通过 **0 佣金**引入更多商家入驻（沉淀自有供给）。
